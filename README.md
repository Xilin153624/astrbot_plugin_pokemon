# 精灵对战插件 for AstrBot

一个类似宝可梦的精灵对战插件，支持捕捉、培养、对战。

## 功能
- 9种属性：火、水、草、飞行、电、地面、机械、暗、光
- 属性克制系统
- 精灵可学习最多4个技能
- 玩家队伍最多6只精灵
- 回合制对战，双方输入技能指令

## 数据文件
存储在 `AstrBot/data/pokemon/` 下：
- `pokedex.json`：精灵图鉴
- `trainers.json`：玩家数据

## 使用命令
- `我的精灵` - 查看自己的精灵队伍
- `选择 N` - 选择第N只精灵出战
- `技能 N` - 在对战中使用第N个技能
- `开始对战` - 发起对战请求
- `接受对战` - 接受对战

## 扩展
你可以自行编辑 `pokedex.json` 添加更多精灵和技能。
